# Artist Website (Flask)

A high-end, minimalist artist website built with Python Flask.

## Features
-   **Server-Side Rendering (SSR)** for SEO optimization.
-   **Dynamic Music Page**: Top tracks, latest releases, and drop countdowns.
-   **Booking System**: Specialized inquiry form.
-   **Responsive Design**: Mobile-friendly navigation and layout.

## Project Structure
-   `app.py`: Main Flask application.
-   `templates/`: HTML files (Jinja2).
-   `static/`: CSS, JS, Images.

## How to Run

1.  **Install Dependencies**:
    ```bash
    pip install flask
    ```

2.  **Start the Server**:
    ```bash
    python app.py
    ```

3.  **Visit**: Open `http://localhost:5000`

## Customization
-   **Styles**: Edit `static/css/styles.css` to change colors/fonts.
-   **Content**: Edit HTML files in `templates/`.
